# UNICANA

Landing page oficial da UNICANA — Cadeia Nacional Produtiva da Cachaça.

## Estrutura

- `index.html`: página principal pronta para publicação
- `LOGO.png`: logotipo usado no site
- `novasfotos/`: imagens principais da nova versão visual
- `*.jfif`: imagens institucionais adicionais usadas na página

## Publicação local

Abra o arquivo `index.html` no navegador para revisar o site.

## Deploy

Este projeto está preparado para publicação no GitHub Pages.
